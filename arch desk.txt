Pour lancer l’application DeskCaisse de manière simple et silencieuse, j’ai créé un fichier start-deskcaisse.vbs,
 que l’on peut placer dans shell:startup pour qu’il se lance automatiquement au démarrage de Windows. 
Ce fichier VBS utilise un chemin absolu pointant vers le fichier start-deskcaisse.bat situé dans le dossier du projet :
C:\Users\HP\Desktop\Main\MyProjects\Deskcaisse\start-deskcaisse.bat.

Le rôle du VBS est de lancer le .bat en arrière-plan sans ouvrir de console visible, 
ce qui permet de démarrer rapidement le backend Laravel, le frontend React, et d’ouvrir automatiquement la PWA dans Chrome maximisée.

Pour les versions futures, je prévois d’utiliser directement les builds Laravel (php artisan build)
 afin d’améliorer la vitesse de chargement des API en environnement de production,
 tout en laissant React en mode développement pour conserver la flexibilité et la rapidité lors du développement.

Il est également possible de consulter les logs du lancement du frontend et du backend 
en utilisant la commande Win + R → %TEMP%, où les fichiers backend_log.txt et frontend_log.txt sont automatiquement créés par le script.
